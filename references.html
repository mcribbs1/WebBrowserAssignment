References & Learning Resources

The following references and sources were used to research, write, and build this website project for IT 3203. They include books, official documentation, and reputable websites related to the history, mechanics, and modern development of web browsers.

  • Berners-Lee, Tim. WorldWideWeb Browser Project, CERN, 1990.
  • Cusumano, Michael A., and David B. Yoffie. Competing on Internet Time: Lessons from Netscape and Its Battle with Microsoft. Free Press, 1998.
  • Mozilla Foundation. Firefox Project Documentation, Mozilla, 2004.
  • NCSA. Mosaic Browser History, University of Illinois, 1993.
  • Shklar, Leon, and Richard Rosen. Web Application Architecture: Principles, Protocols, and Practices. Wiley, 2003.
  • StatCounter. “2024 Browser Market Share.” StatCounter Global Stats, 2024.
  • W3C. Web Storage Specification, World Wide Web Consortium, 2016.
  • IETF. RFC 9110: HTTP Semantics, Internet Engineering Task Force, 2022.
