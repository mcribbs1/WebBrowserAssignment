Key Concepts Explained

HTTP vs. HTTPS

HTTPS encrypts traffic with TLS for privacy and integrity; HTTP does not.

DOM (Document Object Model)

A tree representation of the page that scripts and styles use to affect rendering.

Rendering Engine

Parses HTML and CSS to draw pixels on the screen (e.g., Blink, WebKit, Gecko).

JavaScript Engine

Runs JS code and interacts with the DOM for interactivity (e.g., V8, SpiderMonkey).

Cookies vs. Cache

Cookies store small pieces of site data; cache stores resource copies to load faster.

WebAssembly

A low-level binary format that lets compiled code run efficiently inside browsers.
